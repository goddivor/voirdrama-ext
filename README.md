# VoirDrama — Aniyomi Extension

Extension [Aniyomi](https://aniyomi.org/) / Anikku pour [voirdrama.to](https://voirdrama.to).

## Installation

Télécharge la dernière version de l'APK depuis la page [Releases](https://github.com/goddivor/voirdrama-ext/releases) et installe-la sur ton appareil Android.

## Langue

Français (fr)

## Fonctionnalités

- Parcourir les dramas populaires / dernières MàJ
- Recherche avec filtres (tri, format, langue, année, statut, genres)
- Qualité des miniatures configurable
- Lecteur préféré (Vidmoly, Filemoon, VOE, StreamTape, VK) + qualité préférée
- Support des deep links depuis `voirdrama.to`
